Chatbot Widget - Embed Instructions

Step 1: Update Configuration

Edit chatbot-widget.js and replace your-deployed-app.vercel.app with your actual deployed URL.

Step 2: Embed on Any Website

Add this script tag to any website where you want the chatbot:

<script src="https://arivubot-seven.vercel.app/chatbot-widget.js"></script>

Step 3: Customization Options

You can customize the widget by modifying these variables in the script:

  • Colors: Change #007bff to your brand color
  • Position: Modify bottom/right values
  • Size: Adjust width/height of the chat window

Features

  • ✅ Floating chat button
  • ✅ Expandable chat window
  • ✅ Auto-detects website URL for context
  • ✅ Responsive design
  • ✅ Easy to embed with single script tag

Example Usage

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>Your website content here...</p>
    
    <!-- Add chatbot widget -->
    <script src="https://arivubot-seven.vercel.app/chatbot-widget.js"></script>
</body>
</html>
